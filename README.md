# Library CRUD
A small project where you can register your books.

# Technologies
* Spring Boot
* Spring Data JPA


# Practices
* API REST
* Dependencies Injection
* Error Treatment Response

# How to execute 
* Clone the repository
* Build the project
* Execute Application
* Test API in Postman or Insomnia
